# Deploy Aussie Saver

## Render (recommended)
1. Push this project to GitHub.
2. In Render, click **New +** -> **Blueprint**.
3. Select your repo. Render will detect `render.yaml`.
4. Add env vars in Render dashboard:
   - `RAPIDAPI_KEY` (or per-store keys)
5. Deploy.

## Local run
```bash
npm start
```
